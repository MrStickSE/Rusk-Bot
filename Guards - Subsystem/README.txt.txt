Guards README:

Guards is one of the subsystems that will be applied in the analytical rule-based CLI CUI chatbot AI, Rusk-Bot.
Guards is the "Login and Signup Subsystem". It has 2 different algorithms that are fired depending if the user wants to signup or login.
Required text files :
"user.txt"
"pass.txt"

Write your username in the file user.txt. Write your password for that username with the same index as the username. Make sure to convert the password into the hashed form using encrypter.py.
